# Wasteland (1988) Strategy Guide

## Scope and spoiler policy

This guide covers the original Interplay DOS version of **Wasteland**.

It is written for a first complete playthrough, but it contains story and solution spoilers.

The official manual and the numbered paragraph book are the primary references for controls and game systems.

Names, passwords, item combinations, and local events can vary between releases and installed scenarios.

When a clue conflicts with this guide, trust the text the game gives you and its requested paragraph.

## Overview and objective

Wasteland is a post-nuclear role-playing game set in the American Southwest after the 1998 nuclear exchange.

You command a party of Desert Rangers from Ranger Center.

Your initial assignment is to investigate disturbances west of Ranger Center.

The early trail leads through Highpool, the Agricultural Center, and the Rail Nomads' Camp.

That local investigation expands into the problems of Quartz, Needles, and Las Vegas.

The deeper plot concerns robot attacks, Project Darwin, Irwin John Finster, and the machine intelligence at Base Cochise.

The practical definition of victory is to expose and overcome the forces behind the robot threat.

That means surviving the late-game facilities, dealing with Finster, and stopping **Cochise** at Base Cochise.

Do not mistake an apparently completed town quest for the end of the campaign.

The game regularly offers odd jokes, false leads, and deliberately absurd paragraph-book branches.

Keep following evidence about the robots, bases, keys, and technical equipment.

## The party

A party may hold up to seven people.

Only four slots are for player characters.

The remaining slots can be filled by recruited non-player characters, or NPCs.

The default party is Hell Razor, Angela Deth, Thrasher, and Snake Vargas.

The manual is explicit that they have no hidden divine advantage over custom characters.

A custom party gives better skill coverage; the default party is a valid way to begin immediately.

NPCs can provide abilities, equipment, plot access, or local knowledge that a custom Ranger cannot.

NPCs are autonomous rather than programmable party members.

They may refuse trades or make their own choices in combat.

Do not permanently dismiss an NPC until you know that character is no longer needed.

## Before leaving Ranger Center

1. Inspect every starting Ranger with the number keys.
2. Confirm who has medical, perception, and firearm skills.
3. Equip a usable weapon and compatible ammunition on every combatant.
4. Put a medic and perceptive character near the front of the marching order.
5. Save before entering an unfamiliar town, building, or dungeon.
6. Carry water, ammunition, and cash before treating desert travel as routine.
7. Keep one or two inventory slots free on multiple characters.

## Full controls reference

### Movement and interface

| Input | Function |
|---|---|
| `I` or Up Arrow | Move north / up |
| `J` or Left Arrow | Move west / left |
| `K` or Down Arrow | Move south / down |
| `L` or Right Arrow | Move east / right |
| Arrow keys | Alternative directional movement |
| Mouse | Point the directional arrow, then hold the button to move |
| `Space` | Toggle the party roster view; in combat, show the immediate map |
| `Esc` | Pass time when not cancelling a menu or command |
| Hold `Esc` | Pass time faster |
| Mouse on party | Another way to let time pass |
| `PgUp` | Review earlier messages |
| `PgDn` | Return toward the newest message |
| Number key | View that character's profile |
| `Shift` + number | Open **Use** for that specific character |
| `Ctrl` + `R` | Reorder items or skills in the appropriate character screen |
| `Ctrl` + `A` | In combat, show enemy groups and distances when eligible |
| Up Arrow / `Ctrl` + `K` | Speed combat text scrolling |
| Down Arrow / `Ctrl` + `J` | Slow combat text scrolling |

The game often communicates the important part of a puzzle in its message history.

Use `PgUp` before leaving an area when an NPC, sign, terminal, or discovery gave a clue.

The `Space` key is especially useful in encounters: hide the roster and text temporarily to see the battlefield.

### Macro functions

Macros repeat a recorded series of keystrokes.

1. Press `Ctrl` plus `F1` through `F10` to start recording in that slot.
2. Release the keys after the `REC.MAC.` indicator appears.
3. Enter the command sequence to record.
4. Press `Ctrl` plus the same function key to stop recording.
5. Press that function key later to repeat the sequence.

The manual's example records several `Esc` presses to pass time quickly.

Use macros for safe, repetitive actions rather than irreversible inventory actions.

Erase a macro by holding `Ctrl` and pressing its function key twice.

### Ranger Center commands

| Command | Purpose |
|---|---|
| **Create** | Roll and build a new player character |
| **Delete** | Remove a player character to free a PC slot |
| **Play** | Leave Ranger Center and begin field play |

At creation, press `Space` to reroll attributes.

Confirm the roll, enter a name of up to thirteen letters, choose sex and nationality, then purchase skills.

Use `Ctrl` + `R` during skill selection to restart the allocation if necessary.

### Non-combat command menu

| Command | What it does | Practical use |
|---|---|---|
| **Use** | Uses an item, skill, or attribute in a direction or on the current square | Search, heal, unlock, apply a quest item, or test an obstacle |
| **Enc** | Opens encounter/combat commands outside normal combat | Recruit an NPC, reload several people, or initiate an action efficiently |
| **Order** | Sets marching order | Put Perception, medical help, or a key item carrier where needed |
| **Disband** | Splits the party into up to four groups | Solve one-person passages, scout, bury dead, or use tactical diversions |
| **View** | Switches control between disbanded groups | Continue operating the party that is off-screen or on another map |
| **Save** | Saves the game and can quit to DOS | Preserve major progress before moving on |
| **Radio** | Contacts Ranger Center for promotion review | Use frequently while all eligible Rangers are conscious |
| **Print** | Prints displayed party information | Historical DOS utility; not normally necessary in emulation |

For **Use**, select a Ranger, then select an item, skill, or attribute.

Finally select a direction, or press `Space` to act on the current square.

“That doesn't seem to work” means that method failed; it does not prove the location has no solution.

The manual notes that repeating a skill attempt can sometimes succeed.

**Order** changes exploration order, not immunity from combat damage.

To make an order, select the first character, second character, and so on until the roster is complete.

**Disband** does not stop time for the party left behind.

If asked whether to dismiss an NPC permanently, answer carefully: confirming dismissal removes that NPC forever.

Separated groups reunite automatically when they enter the same square.

### Combat commands

| Command | Function |
|---|---|
| **Run** | Move the full party or one character one space toward or away from an encounter |
| **Use** | Use an item, skill, or attribute in combat |
| **Hire** | Ask a non-hostile NPC to join |
| **Evade** | Make a character harder to hit that round |
| **Attack** | Choose a target and fire mode where available |
| **Weapon** | Ready a different weapon |
| **Load/Unjam** | Reload or repair a jammed weapon; costs a full round |

A single character who successfully runs away becomes disbanded if another group slot is available.

Use **Evade** for an injured character, a melee-only character at range, or a civilian you cannot afford to lose.

Use **Weapon** rather than forcing an unjam when a second loaded gun is available.

Select every party member's action, then confirm with `Y` to execute the round.

Use `N` at confirmation to abandon the selections and reconsider.

During a multi-party encounter, avoid running combats on remote maps unless it is essential.

## Character creation

### Attributes

| Attribute | Meaning in the manual | Build implication |
|---|---|---|
| **ST** — Strength | Power, lifting, moving and breaking things; important in hand-to-hand combat | Best on a brawler or physical problem-solver |
| **IQ** — Intelligence | Determines which skills can be mastered and how many skill points begin available | The most important creation stat for flexible characters |
| **LK** — Luck | Helps finding things, avoiding damage, and hand-to-hand odds | Valuable general insurance |
| **SP** — Speed | Determines quick movement and helps escape tight situations | Helps mobile/scouting characters |
| **AGL** — Agility | Dexterity of gross movement; improves acrobatics, dodging, jumping, and melee | Strong for front-line combatants |
| **DEX** — Dexterity | Fine control for locks, aiming, combat, and thiefly skills | Important for shooters and technical characters |
| **CHR** — Charisma | Likeability and persuasion; affects NPC reactions and trading | Put it on a party spokesperson |

A new character begins with **skill points equal to IQ**.

Higher IQ both unlocks more advanced skills and provides the points to buy them.

A character may learn a skill only when their IQ meets its minimum.

Each additional level of a bought skill doubles its point cost.

### Sex and nationality

Choose sex and one of the five nationalities: U.S., Russian, Mexican, Indian, or Chinese.

The manual presents nationality as a character-choice option rather than a stat bonus system.

Sex matters chiefly for access to gendered bathroom locations.

Do not sacrifice skill coverage for a role-playing preference unless you are deliberately taking a challenge run.

### Recommended starting party

A strong four-PC party uses distinct roles rather than four nearly identical shooters.

| Role | Priorities | Start with |
|---|---|---|
| Field medic | High IQ; solid DEX and SP | Medic, Perception, Clip Pistol, optionally Swim |
| Rifle specialist | Good DEX, AGL, IQ | Rifle, Clip Pistol, Perception, Brawling or Climb |
| Scout / thief | High IQ, DEX, SP, AGL | Perception, Climb, Swim, Picklock, Silent Movement |
| Demolitions / utility | High IQ; good DEX | Demolition, Alarm Disarm, Bomb Disarm, combat skill |

Give **Medic** to at least two characters as soon as practical.

Give **Perception** to more than one character if your skill points allow it.

Every active combatant should have a skill for the weapon they expect to use.

Do not make your only Medic the same Ranger who always stands in front and draws fire.

Do not make every Ranger a specialist with no firearm skill.

### Skill priorities

1. **IQ** is a strategic priority because promotions can turn IQ increases into new skill points.
2. **Medic** prevents a serious wound from becoming a permanent death.
3. **Perception** supports finding concealed objects and noticing abnormalities.
4. **Clip Pistol**, **Rifle**, **SMG**, and **Assault Rifle** make weapon upgrades usable.
5. **Climb** and **Swim** prevent terrain from becoming a dead end.
6. **Picklock**, **Safecrack**, **Alarm Disarm**, and **Bomb Disarm** protect access to rewards.
7. **Demolition** matters for explosive obstacles and should be on a careful, living specialist.
8. **Brawling** or **Pugilism** give a fallback when guns are unavailable or unwanted.

### Improving and ordering skills

Skills improve by use, making repeated legitimate use a cheaper route than buying many levels.

Do not squander scarce initial points buying deep levels of one skill before covering essentials.

When viewing skills, use `Ctrl` + `R` to set their display order.

The skill list must be fully reordered before the new order appears.

## Combat guide

### Read the range before choosing weapons

Hand-to-hand attacks work only at **14 feet or less**.

Contact weapons include fists, knives, axes, and similar melee equipment.

Beyond 14 feet, use a projectile weapon, move, evade, or run.

The manual divides missile weapons into short, medium, and long range.

A pistol can work at a distance, but it is not the right answer to a far-away target.

Use rifles and assault rifles for long-range engagements.

Use SMGs in closer firefights.

Use pistols as reliable early weapons and backups.

### Weapons and ammunition

| Weapon family | Range role | Ammunition / handling |
|---|---|---|
| 1911A1 .45 pistol | Short | Eight-shot .45 clip; effective early sidearm |
| VP91Z 9mm pistol | Short | Eighteen-shot 9mm capacity reduces reload frequency |
| M19 Rifle | Long | Eight-shot 7.62mm clip; single fire |
| M17 Carbine | Long | Ten-shot 7.62mm clip; shorter barrel trades some range accuracy |
| AK-97 | Long | Thirty-shot 7.62mm clip; selective fire |
| M1989A1 | Long | 7.62mm assault-rifle platform; selective fire |
| MAC 17 | Medium | Thirty-shot .45 SMG clip; close firefights |
| Uzi 27 | Medium | Forty-shot 9mm SMG clip; selective fire |
| Grenades / TNT / plastic explosives | Area effect | Automatically supplies another item until the stock is gone |
| LAW rocket | Long anti-armor | Disposable anti-tank weapon; re-equip after firing |

All clip-fed guns need the matching .45, 9mm, or 7.62mm ammunition.

Reloading costs a full combat round, so reload after battles rather than at the next ambush.

A jam loses the ammunition currently in the weapon.

Unjamming in combat costs a full round but automatically reloads it.

### Selective fire

Automatic weapons offer **Single**, **Burst**, and **Autofire**.

| Mode | Ammunition use | Best use |
|---|---:|---|
| Single | 1 bullet | Routine shots, low ammunition, conserving supplies |
| Burst | 3 bullets | Better chance to hit or kill without emptying a clip |
| Autofire | Entire clip | Emergency damage, clustered enemies, decisive rounds |

Autofire improves the chance of hitting a chosen enemy and nearby targets.

It also turns a full magazine into one round of damage.

Do not use it as the default against weak enemies.

Missile combat is simultaneous: killing an attacker this round does not guarantee they will not shoot back this round.

### Explosives

Grenades and similar area-effect weapons can save a party that is being swarmed.

They are auto-loading while inventory remains, unlike LAW rockets.

Explosives damage a broad target area, so protect your own party's position before using them.

Use Demolition for explosive problems and do not assume a large charge is automatically the best choice.

Preserve rockets and high-grade explosives for armored, robotic, or otherwise decisive encounters.

### Healing and injury states

| State | What happens | Correct response |
|---|---|---|
| Lightly wounded | Constitution is reduced but the Ranger can act | Heal when safe; avoid taking needless follow-up damage |
| Unconscious | Constitution is below 1; cannot move | Wait for recovery or seek a hospital / fast medical care |
| Serious | Cannot act and worsens over time | Apply Medic immediately |
| Critical | Worsening serious condition | Stabilize immediately; do not travel for help alone |
| Mortal | Near death | Use the best available medical response at once |
| Comatose | Final stage before death | Emergency stabilization only; time is nearly gone |
| Dead | Permanent loss after saving / map transition | Bury via Disband or recover only by reverting to an earlier save state |

An unconscious Ranger can recover to full health with time.

A seriously wounded Ranger does **not** heal merely by waiting; the sequence is serious, critical, mortal, comatose, then dead.

A successful medical action can stabilize the victim down to the unconscious category, after which time can heal them.

Sending one Ranger across the desert for help does not pause the victim's deterioration.

This is why two Medics are safer than one.

### Promotions by Radio

Experience comes from difficult tasks and hard battles.

Use **Radio** whenever the party has progressed and all candidates are conscious.

There is no cost to requesting a review.

A successful field promotion gives:

- Two automatic points of maximum constitution.
- Two additional attribute points to assign.

Spend promotion points on **IQ** if you want to gain skill points for new skills.

Otherwise, distribute points to shore up combat, survivability, movement, or interaction needs.

An unconscious or injured Ranger is skipped during the promotion procedure.

## Progression walkthrough

This route is intentionally goal-oriented rather than a square-by-square solution.

It preserves discovery, avoids pretending that every release has the same text variant, and keeps you moving toward the main plot.

### 1. Ranger Center: establish a survivable team

- Create a balanced party or inspect the pre-generated Rangers.
- Ensure at least one Ranger starts with a compatible pistol skill and weapon.
- Build toward two Medics and multiple Perception users.
- Buy broad first levels before expensive specialization.
- Set marching order with a perceptive, durable lead character.
- Save before leaving.

Do not leave with every party member carrying the same narrow combat role.

The first map is not a safe tutorial zone; wilderness travel can already punish poor water, ammunition, or injury management.

### 2. Highpool: investigate the first local crisis

Highpool is one of the manual's intended first objectives west of Ranger Center.

- Speak to residents and read all messages before acting.
- Identify the immediate threat to the settlement and the equipment or actions that resolve it.
- Use Perception, Climb, and medical skills where the environment suggests them.
- Recruit useful allies if they are willing, but do not permanently dismiss them casually.
- Take notes on clues, named people, and technical objects.

Highpool teaches the central Wasteland pattern: dialogue, a physical obstacle, a skill check, an item, and a consequence may all be one quest.

### 3. Agricultural Center: follow the settlement leads

The Agricultural Center is the second starting destination named by the manual.

- Treat it as an investigation, not merely a shop stop.
- Talk to the farmers and key figures before searching every structure.
- Address the threat to their food, people, or defenses before assuming a combat solution is required.
- Look for recruitment opportunities and information that points to the Rail Nomads.
- Restock water, medical items, and ammunition if available.

The paragraph book's local descriptions repeatedly frame the farmers as facing a serious, unusual enemy.

That is an early signal that the campaign is moving beyond ordinary bandits.

### 4. Rail Nomads' Camp: learn the factions and use the clues

The Rail Nomads are an early hub with clans, a trading car, a casino, and an oracle / Hobo source of clues.

- Meet the Brakeman and Engineer rather than treating the camp as a single faction.
- Pay attention when one leader reacts negatively to another's name.
- Complete favors for clan contacts and record any passwords, visas, cache directions, and rail landmarks.
- Use the Hobo's statements as clues, but verify them against the situation.
- Trade and restock before the longer town sequence.

The paragraph book contains several alternate faction names and clue variants.

Use the exact clue from your installation rather than relying on a generic password list.

### 5. Quartz: resolve the bandit problem and gather leads

Quartz is a small post-war town with a bandit problem.

- Talk to the town's authorities, merchants, and people connected to the Rail Nomads.
- Present any visa or proof of favor only to the contact your in-game clue identifies.
- Follow directions to caches carefully; a misplaced step can waste time and create danger.
- Look for leads on Needles, prisons, outlaws, and the widening robot story.
- Keep a save before committing to fights in a hideout or hostile building.

Quartz is a good place to improve gear before Needles.

It is also a useful reminder that an item that appears to be a simple card, bracelet, scrap, or pass can be a quest key.

### 6. Needles: cults, prisoners, and the Bloodstaff trail

Needles is larger, suspicious of outsiders, and filled with competing groups.

- Speak first; the town has information that changes how later buildings should be approached.
- Separate civic problems, hostage leads, and cult business in your notes.
- Investigate the Temple of Blood / Mushroom Church material carefully.
- Recover the **Bloodstaff** if the local clues make it your bargaining item.
- Do not hand over unique story objects until the recipient and promised next step are clear.
- Search for the connection between Needles, Darwin, and the robot plot.

The paragraph book directly ties returning the Bloodstaff to information about Max / Finster and a **Sonic Key**.

That trade is a main-story pivot, not just optional treasure hunting.

### 7. Las Vegas: meet Brygo and find the missing expert

Las Vegas is the main urban pivot from regional survival to the machine conspiracy.

- Meet Faran Brygo / the Big Boss rather than joining a side faction blindly.
- Hear the request concerning the missing technical fighter, Finster or the figure named by your version.
- Treat offers from Fat Freddy and the Servants of the Mushroom Cloud as factional, self-interested proposals.
- Explore the church, criminal, and sewer leads methodically.
- Use the Bloodstaff-derived clue and Sonic Key information to reach the relevant route.
- Keep notes on passwords, because their exact values are scenario text rather than universal player knowledge.

Brygo explains that robot attacks are increasing and that the missing figure has technical knowledge relevant to stopping them.

The Las Vegas quest is therefore the bridge to the endgame facilities.

### 8. Darwin: uncover Finster's project and secure the technology

Project Darwin is a dangerous science-base sequence.

- Enter with healthy Rangers, loaded rifles, medical supplies, and room for technical loot.
- Read terminals and personnel material instead of discarding it as flavor text.
- Expect clues about Finster, cloning, experimental creatures, and the history of the larger program.
- Retrieve components only after confirming that they are portable quest equipment rather than expendable junk.
- Be especially cautious around clone technology, access systems, and high-damage encounters.

Darwin explains why the human antagonist, experimental technology, and the escalating robotic danger are related.

It also provides or contextualizes equipment necessary for the late game.

### 9. Base Cochise, Guardian Citadel, and Sleeper Base: finish the machine threat

Late-game locations are not interchangeable detours.

The proper route comes from the maps, terminals, technical components, and keys you have accumulated.

- Confirm which facility your current clues identify before crossing radiation-heavy wilderness.
- Carry the **Sonic Key** separately in your notes from the four Citadel keys.
- Assemble and protect the required technical components; do not sell or drop them for inventory space.
- Use Guardian information and safe-passage clues rather than testing minefields by walking through them.
- In the Sleeper Base, restore or use the base systems as the game directs before expecting the final route to open.
- In Base Cochise, follow the objective of defeating the intelligence, not merely clearing visible guards.

The paragraph book identifies Base Cochise as the home of the deadly robots and names its artificial-intelligence connection.

It also warns that the four Citadel keys are distinct from the Sonic Key.

The endgame rewards careful reading more than brute-force wandering.

## Key items and where they matter

### Weapons and armor

| Item | Why it matters | Guidance |
|---|---|---|
| Ruger / high-quality pistol | Strong early personal firearm | Keep compatible ammunition and pistol skill |
| AK-97 | Common, long-range selective-fire rifle | Equip on an Assault Rifle-trained Ranger; preserve 7.62mm clips |
| M1989A1 | Long-range assault-rifle alternative | Same 7.62mm logistics; selective fire is powerful but expensive |
| M19 Rifle | Long-range single-shot rifle | Useful before assault rifles or for conserving ammunition |
| Uzi 27 / MAC 17 | Medium-range automatic weapons | Give them to SMG-skilled Rangers; watch clip consumption |
| RPG-7 / LAW rockets | Anti-armor and decisive heavy damage | Save for robotic or armored threats; re-equip expendable launchers as needed |
| Grenades / explosives | Area damage | Use against groups, but keep friends outside the effect area |
| Power armor | Major late-game protection | Reserve for frontline Rangers facing facility and robotic danger |

Weapon names and availability can be version- and discovery-dependent.

The tactical rule does not change: match the user's skill, weapon range, and ammunition supply.

### Keys, passes, and access items

| Item / category | Where it matters | Handling rule |
|---|---|---|
| Keys and passcards | Doors, guarded facilities, faction checkpoints | Keep until the related area is fully resolved |
| Visa cards and faction tokens | Rail Nomads / Quartz-style favors | Present only to the named recipient from the clue |
| Blackstar, Quasar, Nova, and Pulsar keys | Guardian Citadel access sequence | Treat as a four-key set; do not confuse with the Sonic Key |
| Sonic Key | Vegas sewer / plot access route | A distinct key made or hidden in connection with the missing expert |
| Bloodstaff | Needles religious quest and information exchange | Do not discard; it unlocks a pivotal lead |
| Passwords and launch codes | Terminals, gates, faction encounters | Write down the exact value the game gives your party |

The paragraph book explicitly warns not to confuse the Sonic Key with the four Citadel keys.

A key item may look small and worthless in a 30-item inventory, but can be irreplaceable.

### Advanced technical equipment

| Item / category | Story use | Handling rule |
|---|---|---|
| Clone technology / clone pods | Darwin's experimental and recovery-related systems | Use only when the game context clearly identifies the procedure |
| Plasma coupler | High-technology destruction / power objective in the late plot | Preserve it for the stated mission; it is not ordinary loot |
| Power converter | Late-game power and system restoration chain | Keep with other technical parts until the system accepts it |
| Technical components | Sleeper Base / Cochise progression | Do not sell, drop, or scatter among inaccessible parties |
| Personnel files and maps | Reveal history, locations, and solutions | Read and record names, directions, and codes |

The paragraph book calls out the plasma coupler as specialized equipment for a high-radiation objective.

Even when a component has no immediate combat value, it may be the actual gate to the ending.

This is a **schematic orientation map**, not a replacement for the game's exact terrain grid.

The manual places Ranger Center in the south of the operating area and the Grand Canyon to the north.

It names Highpool, the Agricultural Center, Rail Nomads' Camp, Quartz, Needles, and Las Vegas as important destinations.

Use the actual in-game map, local directions, and discovered landmarks to navigate.

The important planning principle is a south-to-north escalation: starting settlements first, major towns next, then the remote technical facilities.

### Ranger Center: functional layout

```text
[Character roster]
        |
        +-- Create: roll, name, sex, nationality, skills
        |
        +-- Delete: remove a PC slot occupant
        |

```

Ranger Center is not a conventional dungeon.

Its key function is character creation, roster management, promotion contact by radio, and the start of the overworld route.

```text
           +-- route toward the larger machine plot
```

The city is deliberately political.

Meet Brygo, listen to competing claims, and do not assume the first reward offer advances the main quest.

## Tips, tricks, and known exploits

### Saving is part of strategy

Wasteland is a dynamic game that permanently updates locations and party state.

Taking an item changes the world; leaving and returning does not restore it.

Entering a new location saves changes to the old and new locations.

Save after resolving a major puzzle, obtaining a unique item, recruiting someone important, or surviving a hard battle.

The manual's death recovery trick is historically specific:

1. If a character dies, **do not** enter a new location.
2. Do not save the game.
3. Reboot or restart before permanent state is written.
4. Resume from the last save or map-change state.

This can restore the character but loses acquisitions since that state.

Use it as a safety net, not a substitute for regular saving.

### Inventory discipline

Each character can carry up to thirty items.

Picking up an item means keeping it or dropping it.

Dropped items vanish permanently.

Therefore:

- Keep quest objects, keys, passes, maps, and technical parts.
- Do not discard an unfamiliar unique item merely because it has no immediate use.
- Spread ammunition, water, and medical items across more than one conscious Ranger.
- Reserve a little capacity before entering a facility likely to contain quest equipment.
- Trade equipment deliberately; NPCs may refuse to give items back.

### Skill practice

The manual says skills improve as they are used.

Use skills legitimately and often when a situation supports them.

Examples include applying medical skills after injuries, using weapon skills in actual fights, and attempting relevant environmental actions.

Do not spend all creation points on high levels before establishing broad party coverage.

Repeated use is generally cheaper than repeatedly buying doubled-cost levels.

### Money and equipment strategy

Use **Pool** on the first character screen to centralize cash before an expensive purchase.

Use **Div Cash** to distribute funds evenly afterward when appropriate.

Cash is useful for supplies, services, gambling, and local opportunities, but a quest key can be worth more than any purchase.

Sell obvious duplicates only after confirming they are not special variants or required components.

Gambling can provide money, but it should not be your sole economic plan.

The safest wealth strategy is to complete settlement quests, avoid wasteful ammunition use, and convert surplus ordinary loot into supplies.

### Desert survival

The manual describes the desert as dangerous because of heat, lack of water, radiation zones, bandits, and mutated animals.

Carry canteens or water before long overland trips.

Resupply in towns instead of treating every excursion as a one-way sprint.

Avoid unknown radiation zones until you have the protection, health, and reason to cross them.

A highlighted maximum-constitution value indicates a life-threatening illness such as radiation poisoning.

That condition does not disappear simply because time passes; seek medical help.

### Tactical habits

- Reload after every encounter, not in the opening round of the next one.
- Keep a backup weapon equipped or ready for a jam.
- Use `Enc` outside combat to reload multiple people efficiently.
- Reserve full auto and rockets for threats that justify their cost.
- Evade with unarmed, wounded, or exposed characters.
- Use marching order to put Perception and situational tools forward.
- Radio for promotions frequently, but only when candidates are conscious.
- Do not split the party around a medical emergency unless someone remaining can stabilize the patient.

## Paragraph book note

The separate paragraph book is both a lore device and a copy-protection-era game mechanism.

The game tells you to read a numbered paragraph when a location, conversation, terminal, or event reaches that reference.

The numbered entry supplies the scene text, clue, outcome, or sometimes a joke and false lead.

Read the specific paragraph the game requests, then return to play.

Do not read the entire book in advance if you want the intended mystery and spoiler structure.

This guide intentionally does **not** reproduce all paragraph text.

Instead, it explains the purpose of the book and summarizes only the plot-relevant guidance needed for strategic planning.

## Quick endgame checklist

Before committing to the northern technical endgame, confirm the following.

- Your active party has at least two medical-capable Rangers.
- Every shooter has a trained weapon, spare ammunition, and a backup plan for jams.
- You have preserved keys, passcards, the Sonic Key, and the distinct Citadel-key set.
- You have not sold, dropped, or isolated technical components such as the power converter or plasma-coupler-related equipment.
- Your party has protection and healing supplies suitable for radiation and robotic encounters.
- You have read terminal, map, and personnel clues from Darwin and the intervening facilities.
- You know the current objective is to reach and defeat Cochise, not simply to loot the next base.
- You saved immediately before entering the major late-game location.

## Final advice

Wasteland rewards notes, restraint, and a party built for more than combat.

Talk before shooting.

Read the message log before guessing.

Preserve every unusual object until its story is finished.

Keep the Medics alive.

Save before a new location.

Then take the Rangers north, follow the evidence, and make certain Cochise does not survive to direct another robot war.
